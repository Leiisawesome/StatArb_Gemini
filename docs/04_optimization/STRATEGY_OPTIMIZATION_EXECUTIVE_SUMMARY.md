## 📈 KEY PERFORMANCE METRICS

### **Risk-Adjusted Returns**
- **Sharpe Ratio**: Return per unit of risk (primary metric)
- **Sortino Ratio**: Return per unit of downside risk
- **Calmar Ratio**: Return per unit of maximum drawdown
- **Information Ratio**: Excess return per unit of active risk

### **Risk Metrics**
- **Maximum Drawdown**: Largest peak-to-trough decline
- **Value at Risk (VaR)**: 95% confidence loss threshold
- **Conditional VaR**: Expected loss beyond VaR
- **Drawdown Duration**: Time to recover from drawdown

### **Trading Metrics**
- **Win Rate**: Percentage of profitable trades
- **Profit Factor**: Gross profit / Gross loss
- **Average Win/Loss Ratio**: Average win / Average loss
- **Trade Frequency**: Number of trades per month

### **Regime Performance**
- **Regime Sharpe**: Sharpe ratio in each regime type
- **Regime Consistency**: Performance variation across regimes
- **Regime Adaptability**: Automatic parameter adjustment
- **Regime Coverage**: Positive returns in multiple regimes
