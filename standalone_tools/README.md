# Standalone Tools and Archived Files

This directory contains standalone scripts and archived documentation that are independent of the main new_structure system.

## Tools

### clickhouse_pair_screening.py
A comprehensive standalone ClickHouse-based pair screening script with advanced analytics features:
- High-performance correlation and cointegration analysis
- Regime-aware screening with volatility analysis
- Liquidity profile assessment
- Transaction cost estimation
- Visualization capabilities

**Usage**: Can be run independently with its own configuration for specialized screening tasks.
